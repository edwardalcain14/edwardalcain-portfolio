## Deployment

This portfolio is designed to be deployed using **GitHub Pages**.

To publish:

1. Push the project to GitHub.
2. Navigate to **Settings → Pages**.
3. Under **Build and Deployment**, select:
   - **Source:** Deploy from a branch
   - **Branch:** `main`
   - **Folder:** `/ (root)`
4. Save your changes.
5. GitHub will generate a live URL for your portfolio.
